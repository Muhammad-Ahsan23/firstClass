	- Virtual DOM kya karta hai?

	. Virtual DOM ek lightweight copy hota hai real DOM ka.

	- Jab koi change hota hai:

	. Pehle change Virtual DOM mein hota hai
	
	Phir React purane Virtual DOM aur naye Virtual DOM ko compare karta hai (isko kehte hain Diffing)
	Sirf jo parts change hue hote hain, unko real DOM mein update karta hai.

	-Real DOM = original printed book 📕
	-Virtual DOM = us book ka photocopy 📄

	Short Summary:

	Virtual DOM ek smart system hai jo real DOM ko directly bar bar update nahi karta, balkay pehle changes ko memory mein handle karta hai aur phir 	sirf necessary updates real DOM mein karta hai.

- useEffect Hooks


	.Remove SideEffect inside Component
	.Remove SideEffect from outSide Component
	.use to fetchdata.
	
	basic use Of useEffect	

	.Prevent Extra rendring of Component on State.

	- useEffect kya hota hai?

		.useEffect ek hook hai jo React me side effects handle karne ke liye use hota hai

	- Side effects ka matlab:

	API se data lana (fetch)
	DOM update karna
	Timer lagana
	console log, etc.

	Example:
		. Remove Side Effects inside Component
		// Wrong
		function App() {
  			fetch("api-url")   // direct call (wrong)

  			return <h1>Hello</h1>
		}
		
	- Ye har render pe run hoga (problem create karega)
	
	- “Remove Side Effects from outside Component”
		
	//  Wrong
	fetch("api-url")

	function App() {
 	 return <h1>Hello</h1>
	}

	
	- Correct way = useEffect

	import { useEffect } from "react";

	function App() {
 	useEffect(() => {
    		console.log("Component load hua");
  	}, []);

  	return <h1>Hello</h1>;
	}
	
	.Ye sirf 1 baar run hoga (on mount)

- Basic use of useEffect

	Run on every render
	
	1. useEffect(() => {
  		console.log("Har render pe run hoga"); // call everytime
	});

	2. useEffect(() => {
  		console.log("Sirf 1 baar"); //Call only once
	}, []);

	3. useEffect(() => {
  		console.log("Count change hua"); // call on single state
	}, [count]);

	4. useEffect(() => {
  		console.log("states change huin"); // call on changeing both state
	}, [state1,state2]);

	5. useEffect(() => {
  		console.log("states change huin"); // call on changeing props
	}, [prop1,prop2]);


- useState hooks
	
	useState ek hook hai jo hamain component me data (state) store aur update karne deta hai

- Basic Syntax

	const [state, setState] = useState(initialValue);
	state = current value
	setState = update karne ka function
	initialValue = starting value

	Example: 

	import { useState } from "react";

	function Counter() {
 		 const [count, setCount] = useState(0);

 	 return (
  	  <div>
     		 <h1>Count: {count}</h1>
	
    		  <button onClick={() => setCount(count + 1)}>
    		    Increase
    		  </button>

   		   <button onClick={() => setCount(count - 1)}>
    		    Decrease
    		  </button>
  	  </div>
  	);
	}
	
export default Counter;

Important Points

- State change → re-render

-Jab bhi setState call hota hai → component dubara render hota hai
